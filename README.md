# DPLS: A power component detection dataset for UAV-based transmission line inspection
This repo stores the dataset of the paper SR-HGDet: Small-Scale Power Transmitting Components Detection in UAV Imagery Based on Super-Resolution and Hypergraph.
## Description
The DPLS dataset is built based on multiple authoritative public datasets, including Tower, CPLID, and InsD. On the basis of the original public data, we strictly screened valid image samples, and completed fine-grained re-annotation of power transmission component targets. In addition, the dataset also integrates many real UAV inspection images captured by actual power line patrol missions.
